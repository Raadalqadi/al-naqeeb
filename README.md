## Overview

Al-Naqeeb is a multi-page corporate platform built to present company information in a polished, scalable, and search-friendly way. The project combines a modern frontend experience with a custom backend and content management workflow to keep the site maintainable after launch.

## Project Snapshot

- Role: Full-Stack Developer
- Worked through: Becon Agency
- Focus: Frontend, backend, dashboard workflow, deployment, and production delivery
- Live website: https://www.alnaqeeb.com/
